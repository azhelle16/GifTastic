ABOUT THIS PROJECT

The goal of this project is to collect GIFs using AJAX based from the button selected.

You can also add new topics to pull out GIFs from by providing data in the line and hit the submit button or press enter.

For every selection of the same button, 10 additional GIFs will be added.

The user can also add a GIF through the FAVORITES section. The added GIF can also be removed by clicking on the REMOVE button

The user can also download a GIF.
